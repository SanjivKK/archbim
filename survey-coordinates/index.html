Features

Everything you need for
geo-referenced BIM

Two focused commands that handle the full coordinate workflow — from capturing positions to relocating elements.

📍

Capture Coordinates

Reads each element's real-world location using the project survey point and writes Northing & Easting as schedulable shared parameters.

🔄

Relocate from Params

Edit a coordinate value in a schedule or properties panel — then run one command to physically move that element to its new position.

🛡️

Smart Change Detection

Only elements whose coordinates have actually been edited get moved. Unchanged elements are skipped, preventing accidental mass-repositioning.

🌿

Site & Planting

Works across both Revit categories — PHE assets (manholes, gullies, inspection chambers) and landscape elements (trees, shrubs, ground cover).

📐

Project Units Aware

Coordinates are expressed in the project's active length unit — metres, feet, or millimetres — with no manual conversion required.

🔍

Post-Move Visibility

After relocating, moved elements are automatically selected and highlighted in the view so you can immediately verify their new positions.

Workflow

Three steps, start to finish

A simple, repeatable process that fits naturally into your existing Revit workflow.

01

Set up shared parameters

Add the provided "Northing" and "Easting" shared parameters to the Site and Planting categories in your project template.

→
02

Capture coordinates

Run Update Parameter Values. The add-in reads every element's position and writes its survey coordinates instantly.

→
03

Relocate from a schedule

Edit a Northing or Easting value in any Revit schedule or properties panel, then run Update Element Locations to move it.
