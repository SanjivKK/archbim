What it does

Everything you need at concept stage

Stop guessing core sizes. Core Layout applies UK residential regulations and gives you correct dimensions and 3D volumes in seconds.

🛗

Regulation-Sized Lifts

Lift shaft dimensions calculated to UK residential standards — dimensions update live as you change storey count or building type.

🪜

Escape Stair Geometry

Stair core widths and lengths set to comply with fire escape requirements. Single and twin stair configurations supported.

🔲

Service Shafts

Horizontal and vertical service shaft volumes included in the core package — sized for typical M&E distribution in residential blocks.

🧊

Live 3D Volumes

Three.js 3D model updates in real time as you adjust inputs. Rotate, zoom, and inspect your core package directly in the browser.

⚡

Instant Results

Change floor count, building type, or configuration and the core dimensions and 3D model update immediately — no waiting, no export step.

🌐

No Install Needed

Runs entirely in any modern browser. No Revit licence required. Free to use at concept and feasibility stage on any device.
