Features

From linked MEP model
to placed, cut, and tagged

Wall Sleeve Cuts handles the full sleeve workflow — detection, placement, wall cutting, marking, and scheduling — without a single manual step.

🔍

Automatic Intersection Detection

Every duct, pipe, and cable tray in the linked MEP model is tested against every wall face using precise geometric intersection — no manual selection, no misses.

✂️

Wall Cutting

Placed sleeves cut their host walls automatically using Revit's void geometry API. The opening is visible immediately in the 3D view — no separate step required.

🏷️

Smart Mark Generation

Each sleeve receives a unique Mark value that encodes the service type and MEP system name — making sleeves easy to identify, filter, and coordinate across disciplines.

D-ME-001 D-MS-002 P-HW-001 CT-001
D · P · CT  =  Duct · Pipe · Cable Tray
📐

Clearance Control

Set top, bottom, and side clearances in millimetres before each run. Fire-rated walls automatically receive an additional clearance margin. Cable trays use asymmetric clearances.

📊

Auto-Populated Schedule

A Wall Sleeve Schedule is created automatically. Every sleeve is tagged with service type, system name, linked model, dimensions, clearances, and host wall ID — ready for coordination.

🔗

Linked Model Picker

If the project has multiple linked models, a clear dialog lists all loaded links so you can choose the MEP model without clicking elements in the viewport.

🎯

Scope Selection

Run on the entire project or limit to the currently active 3D view — useful for checking individual levels or running on a sub-set of walls during coordination.

⬛

Insulation Aware

If ducts or pipes have insulation applied in the linked model, the sleeve dimensions automatically account for it — the sleeve width and height include the insulation thickness.

⏩

Zoom & Select After Placement

After each run, choose to zoom directly to all placed sleeves in your 3D view, or open the Wall Sleeve Schedule — both accessible from a single post-run dialog.
